## ğŸ› ï¸ Installation

### Prerequisites
- Python 3.9 or higher
- pip (Python package manager)
- Ollama (for AI features)

### Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/koushal13/MarketPulse-AI.git
cd MarketPulse-AI

# 2. Install dependencies
pip install -r requirements.txt

# 3. Install and start Ollama
# macOS/Linux:
brew install ollama  # or download from https://ollama.ai
ollama serve

# Pull the AI model (in a new terminal):
ollama pull llama3.2:1b

# 4. Set up API keys (create .env file)
cp .env.example .env
# Edit .env and add your API keys (see Configuration section)

# 5. Run the application
python -m streamlit run app_v2.py --server.port 8505
```

**Access the app:** Open your browser to `http://localhost:8505`

## ğŸ†˜ Troubleshooting

### "Import errors" when running
Make sure all dependencies are installed:
```bash
pip install -r requirements.txt
```

### "API key not configured" warnings
Check that your `.env` file exists and contains valid API keys.

### Model download is slow
The HuggingFace model (~500MB) downloads on first run. This is normal and only happens once.

### Rate limit errors
- You've hit the daily API limit
- Wait 24 hours or upgrade to a paid API plan
- The system will use available data sources

### No news articles found
- The stock symbol might be too new or unlisted
- Try a more popular ticker (AAPL, TSLA, MSFT, etc.)
- Check the NewsAPI dashboard for your usage
